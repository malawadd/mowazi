# Moeazi Landing Motion Script

## Hero Overview
Show the real overview surface immediately. The user should see that Moeazi is already a real operating product with readable health, balances, and execution controls.

## Walkthrough Tour
Move across the real operating pages as a fast guided tour. Show that deposits, activity, risk, and withdrawals each answer a different operating question.

## Trust Model
Translate the system model into a readable motion diagram. Make it obvious that the worker decides, Convex signs and records, and separate venue wallets keep responsibilities legible.

## CTA Context
The landing page itself will carry the CTA copy. These videos stay silent and product-led, so the motion works inline and on loop without needing narration.
